# 第6章: 連続型分布と標本分布 演習問題

## 【大問1】機器の測定モデルと確率分布（一様・コーシー・指数・ガンマ・対数正規）

物理実験の測定誤差や、製造される部品の寿命など、様々な現象は連続型確率分布を用いてモデル化される。以下の問いに答えよ。

### [問題]
1. **（連続一様分布・コーシー分布）** 原点から距離 $L=1$ 離れた直線状のスクリーンに対して、レーザーをランダムな角度 $\Theta$ で照射する。$\Theta$ は区間 $(-\pi/2, \pi/2)$ の**連続一様分布**に従うとする。スクリーン上のレーザーの到達位置を $X = \tan \Theta$ とおく。
   (a) $X$ の累積分布関数 $F_X(x)$ および確率密度関数 $f_X(x)$ を求めよ。
   (b) この分布は**コーシー分布**と呼ばれる。コーシー分布の期待値が存在しない理由を簡潔に説明せよ。

2. **（指数分布・ガンマ分布）**
   ある電子部品の寿命 $T$ はパラメータ $\lambda$ の**指数分布**に従う。
   (a) $T$ の積率母関数 $M_T(s) = E[e^{sT}]$ を求めよ（ただし $s < \lambda$ とする）。
   (b) 独立に同じ指数分布に従う $n$ 個の部品を順次使用したときの合計寿命 $S_n = \sum_{i=1}^n T_i$ が従う分布（**ガンマ分布**）の積率母関数を求め、その確率密度関数 $f_{S_n}(x)$ を記せ。

3. **（対数正規分布）**
   ある粒子の直径 $Y$ について、その自然対数 $W = \log Y$ が**正規分布** $N(\mu, \sigma^2)$ に従うとき、$Y$ は**対数正規分布**に従うという。
   (a) 正規分布の積率母関数の性質を利用して、$Y$ の期待値 $E[Y]$ を $\mu, \sigma^2$ を用いて表せ。

---

### 【大問1 解答解説】

#### 1. 連続一様分布・コーシー分布の解説
**(a) 確率密度関数の導出**
$\Theta \sim U(-\pi/2, \pi/2)$ より、$\Theta$ の確率密度関数は $f_\Theta(\theta) = 1/\pi$ である。
累積分布関数 $F_X(x)$ は以下のように計算できる。
$$F_X(x) = P(X \le x) = P(\tan \Theta \le x) = P(\Theta \le \arctan x)$$
$$F_X(x) = \int_{-\pi/2}^{\arctan x} \frac{1}{\pi} d\theta = \frac{1}{\pi} \left( \arctan x - \left(-\frac{\pi}{2}\right) \right) = \frac{\arctan x}{\pi} + \frac{1}{2}$$
これを $x$ で微分して確率密度関数 $f_X(x)$ を得る。
$$f_X(x) = \frac{d}{dx} F_X(x) = \frac{1}{\pi(1+x^2)}$$

**(b) 期待値が存在しない理由**
期待値の定義式 $E[X] = \int_{-\infty}^{\infty} x f_X(x) dx = \int_{-\infty}^{\infty} \frac{x}{\pi(1+x^2)} dx$ を考える。
この広義積分が収束するためには、$\int_{0}^{\infty} |x f_X(x)| dx$ が有限値をとる（絶対収束する）必要がある。
しかし、$\int_{0}^{R} \frac{x}{\pi(1+x^2)} dx = \frac{1}{2\pi} \log(1+R^2) \to \infty \quad (R \to \infty)$ となり発散するため、コーシー分布の期待値は存在しない。

#### 2. 指数分布・ガンマ分布の解説
**(a) 積率母関数**
指数分布の確率密度関数は $f(t) = \lambda e^{-\lambda t} \quad (t \ge 0)$。
$$M_T(s) = \int_{0}^{\infty} e^{st} \lambda e^{-\lambda t} dt = \lambda \int_{0}^{\infty} e^{-(\lambda - s)t} dt$$
$s < \lambda$ の条件のもとで積分すると、
$$M_T(s) = \lambda \left[ -\frac{e^{-(\lambda - s)t}}{\lambda - s} \right]_0^\infty = \frac{\lambda}{\lambda - s}$$

**(b) ガンマ分布への拡張**
独立な確率変数の和の積率母関数は、それぞれの積率母関数の積となる。
$$M_{S_n}(s) = \prod_{i=1}^n M_{T_i}(s) = \left( \frac{\lambda}{\lambda - s} \right)^n = \left( 1 - \frac{s}{\lambda} \right)^{-n}$$
これはパラメータ $(n, \lambda)$ のガンマ分布の積率母関数に一致する。
したがって、$S_n$ の確率密度関数は次のように表される（$x \ge 0$）。
$$f_{S_n}(x) = \frac{\lambda^n}{\Gamma(n)} x^{n-1} e^{-\lambda x}$$
（※自然数 $n$ の場合、$\Gamma(n) = (n-1)!$ であり、これはアーラン分布とも呼ばれる。）

#### 3. 対数正規分布の解説
**(a) 期待値の導出**
$Y = e^W$ であり、$W \sim N(\mu, \sigma^2)$ である。
$Y$ の期待値は $E[Y] = E[e^W]$ と書ける。これは変数 $W$ の積率母関数 $M_W(t) = E[e^{tW}]$ において、$t=1$ を代入した値に等しい。
正規分布 $N(\mu, \sigma^2)$ の積率母関数は $M_W(t) = \exp\left(\mu t + \frac{\sigma^2 t^2}{2}\right)$ であるため、$t=1$ を代入して、
$$E[Y] = \exp\left(\mu + \frac{\sigma^2}{2}\right)$$

## 【大問2】試験成績の分析（正規・2変量/多変量正規・混合正規）

ある大学の学生の試験成績データを用いて、得点の分布や科目間の相関構造を分析する。

### [問題]
1. **（混合正規分布）**
   統計学の受講生全体における成績 $Y$ を考える。全体の割合 $1-p$ を占める文系学生の成績は正規分布 $N(\mu_1, \sigma_1^2)$ に、割合 $p$ を占める理系学生の成績は正規分布 $N(\mu_2, \sigma_2^2)$ に従う。
   (a) 成績 $Y$ の期待値 $E[Y]$ を求めよ。
   (b) 成績 $Y$ の分散 $V[Y]$ を導出せよ。

2. **（2変量正規分布）**
   学生の数学の成績を $X_1$、物理の成績を $X_2$ とする。これらは**2変量正規分布**に従い、期待値ベクトル $\boldsymbol{\mu} = (\mu_1, \mu_2)^T$、分散共分散行列 $\Sigma = \begin{pmatrix} \sigma_1^2 & \rho\sigma_1\sigma_2 \\ \rho\sigma_1\sigma_2 & \sigma_2^2 \end{pmatrix}$ をもつ（$\rho$ は相関係数）。
   (a) 数学の成績が $X_1 = x_1$ であったときの、物理の成績 $X_2$ の条件付き期待値 $E[X_2 | X_1 = x_1]$ および条件付き分散 $V[X_2 | X_1 = x_1]$ を求めよ。

3. **（多変量正規分布）**
   $n$ 科目の成績ベクトル $\boldsymbol{X} = (X_1, \dots, X_n)^T$ が**多変量正規分布** $N_n(\boldsymbol{\mu}, \Sigma)$ に従うとする。
   (a) この確率密度関数 $f(\boldsymbol{x})$ の式を記せ。
   (b) 確率密度が一定となる等高面はどのような図形になるか答えよ。

---

### 【大問2 解答解説】

#### 1. 混合正規分布の解説
**(a) 期待値**
$Y$ の確率密度関数は $f_Y(y) = (1-p)f_1(y) + pf_2(y)$ と表せる。期待値の線形性より、
$$E[Y] = (1-p)E[X_1] + pE[X_2] = (1-p)\mu_1 + p\mu_2$$

**(b) 分散**
公式 $V[Y] = E[Y^2] - (E[Y])^2$ を用いる。
まず2次モーメント $E[Y^2]$ を求める。各群の分散公式 $\sigma^2 = E[X^2] - \mu^2$ より $E[X^2] = \sigma^2 + \mu^2$ だから、
$$E[Y^2] = (1-p)(\sigma_1^2 + \mu_1^2) + p(\sigma_2^2 + \mu_2^2)$$
次に期待値の2乗を計算する。
$$(E[Y])^2 = \{(1-p)\mu_1 + p\mu_2\}^2 = (1-p)^2\mu_1^2 + 2p(1-p)\mu_1\mu_2 + p^2\mu_2^2$$
分散はこれらの差となる。
$$V[Y] = (1-p)\sigma_1^2 + p\sigma_2^2 + (1-p)\mu_1^2 + p\mu_2^2 - \{ (1-p)^2\mu_1^2 + 2p(1-p)\mu_1\mu_2 + p^2\mu_2^2 \}$$
$\mu$ を含む項だけを整理する。$(1-p) - (1-p)^2 = p(1-p)$、$p - p^2 = p(1-p)$ より、
$$p(1-p)\mu_1^2 + p(1-p)\mu_2^2 - 2p(1-p)\mu_1\mu_2 = p(1-p)(\mu_1 - \mu_2)^2$$
よって、分散は
$$V[Y] = (1-p)\sigma_1^2 + p\sigma_2^2 + p(1-p)(\mu_1 - \mu_2)^2$$
（※群内分散の重み付き和に、群間分散 $p(1-p)(\mu_1 - \mu_2)^2$ が足される形になる）

#### 2. 2変量正規分布の解説
**(a) 条件付き期待値と分散**
2変量正規分布の重要な性質として、条件付き分布もまた正規分布になる。
* **条件付き期待値**（回帰直線）: 
    $E[X_2 | X_1 = x_1] = \mu_2 + \rho \frac{\sigma_2}{\sigma_1}(x_1 - \mu_1)$
* **条件付き分散**: 
    $V[X_2 | X_1 = x_1] = \sigma_2^2(1-\rho^2)$
（※与えられた情報の相関 $\rho$ が強いほど、条件付き分散は小さくなり予測精度が上がる）

#### 3. 多変量正規分布の解説
**(a) 確率密度関数**
行列式を $|\Sigma|$、逆行列を $\Sigma^{-1}$ とすると、
$$f(\boldsymbol{x}) = \frac{1}{(2\pi)^{n/2} |\Sigma|^{1/2}} \exp\left( -\frac{1}{2} (\boldsymbol{x} - \boldsymbol{\mu})^T \Sigma^{-1} (\boldsymbol{x} - \boldsymbol{\mu}) \right)$$

**(b) 等高面の図形**
指数部分に現れる二次形式 $(\boldsymbol{x} - \boldsymbol{\mu})^T \Sigma^{-1} (\boldsymbol{x} - \boldsymbol{\mu})$ はマハラノビス距離の2乗と呼ばれる。これが定数となる領域は、$n$ 次元空間において**多次元楕円体（超楕円体）**となる。

## 【大問3】標本分布とベータ分布（カイ二乗・t・F・非心分布・ベータ）

統計的推測における検定量の構成や、確率モデルの事前分布の仮定において重要な各種分布の性質を確認する。

### [問題]
1. **（カイ二乗分布）**
   標準正規分布に独立に従う確率変数 $Z_1, Z_2, \dots, Z_k \stackrel{i.i.d.}{\sim} N(0,1)$ を考える。
   (a) 標準正規分布の確率密度関数を用いて、$Y = Z_1^2$ の積率母関数 $M_Y(t)$ を求めよ。
   (b) 平方和 $W = \sum_{i=1}^k Z_i^2$ が**自由度 $k$ のカイ二乗分布**に従うことを示せ。

2. **（t分布・F分布・非心分布）**
   (a) 自由度 $k$ のカイ二乗分布に従う変数 $V$ と、標準正規分布に従う $Z$（互いに独立）を用いて、**自由度 $k$ のt分布**に従う変数 $T$ を定義せよ。同様に、**F分布**の定義式も示せ。
   (b) 実務上の検定（例：対応のないt検定）において、帰無仮説が偽である場合、検定量は**非心t分布**や**非心F分布**に従う。これらの非心分布は、(a)の定義における構成要素をどのように変更して生じるものか簡潔に説明せよ。

3. **（ベータ分布）**
   ある事象の成功確率 $p \ (0 < p < 1)$ の事前分布として**ベータ分布** $Be(\alpha, \beta)$ を仮定する。確率密度関数は $f(p) = \frac{p^{\alpha-1}(1-p)^{\beta-1}}{B(\alpha, \beta)}$ である。
   (a) ベータ関数の性質 $B(\alpha, \beta) = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}$ およびガンマ関数の性質 $\Gamma(x+1) = x\Gamma(x)$ を用いて、$p$ の期待値 $E[p]$ を導出せよ。

---

### 【大問3 解答解説】

#### 1. カイ二乗分布の解説
**(a) 自由度1のカイ二乗分布のMGF**
$Y = Z_1^2$ の積率母関数は以下のように計算できる。
$$M_Y(t) = E[e^{tZ_1^2}] = \int_{-\infty}^{\infty} e^{tz^2} \frac{1}{\sqrt{2\pi}} e^{-z^2/2} dz = \int_{-\infty}^{\infty} \frac{1}{\sqrt{2\pi}} e^{-\frac{1}{2}(1-2t)z^2} dz$$
積分が収束する $t < 1/2$ において、$u = \sqrt{1-2t}z$ と変数変換すると $du = \sqrt{1-2t}dz$ となる。
$$M_Y(t) = \frac{1}{\sqrt{1-2t}} \int_{-\infty}^{\infty} \frac{1}{\sqrt{2\pi}} e^{-u^2/2} du = (1-2t)^{-1/2}$$
（※このとき積分部分は標準正規分布の全確率となり1である）

**(b) 自由度 k のカイ二乗分布**
$W = \sum_{i=1}^k Z_i^2$ は独立な確率変数の和なので、積率母関数は積となる。
$$M_W(t) = \prod_{i=1}^k M_{Z_i^2}(t) = \{ (1-2t)^{-1/2} \}^k = (1-2t)^{-k/2}$$
これはパラメータ $(k/2, 1/2)$ のガンマ分布、すなわち自由度 $k$ のカイ二乗分布の積率母関数と一致する。

#### 2. t分布・F分布・非心分布の解説
**(a) t分布とF分布の定義**
* **t分布**: $T = \frac{Z}{\sqrt{V/k}}$
    （分子が標準正規分布、分母がカイ二乗変数をその自由度で割ったものの平方根）
* **F分布**: 独立なカイ二乗変数 $V_1 \sim \chi^2(k_1), V_2 \sim \chi^2(k_2)$ に対して、$F = \frac{V_1/k_1}{V_2/k_2}$

**(b) 非心分布への拡張**
* **非心t分布**: (a)の定義において、分子の $Z$ が標準正規分布 $N(0,1)$ ではなく、平均が $0$ ではない正規分布 $N(\delta, 1)$ に従う場合に生じる。この $\delta$ を非心度と呼ぶ。
* **非心F分布**: 分子の $V_1$ が、中心が $0$ でない正規分布の平方和（非心カイ二乗分布）から構成されている場合に生じる。
検定において「真の差が存在する（帰無仮説が偽）」ときの統計量の分布として現れ、検出力の計算に用いられる。

#### 3. ベータ分布の解説
**(a) 期待値の導出**
$$E[p] = \int_{0}^{1} p \cdot \frac{p^{\alpha-1}(1-p)^{\beta-1}}{B(\alpha, \beta)} dp = \frac{1}{B(\alpha, \beta)} \int_{0}^{1} p^{\alpha}(1-p)^{\beta-1} dp$$
積分部分はパラメータ $\alpha+1, \beta$ のベータ関数の定義そのものであるから $B(\alpha+1, \beta)$ となる。
$$E[p] = \frac{B(\alpha+1, \beta)}{B(\alpha, \beta)} = \frac{\frac{\Gamma(\alpha+1)\Gamma(\beta)}{\Gamma(\alpha+\beta+1)}}{\frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}} = \frac{\Gamma(\alpha+1)}{\Gamma(\alpha)} \cdot \frac{\Gamma(\alpha+\beta)}{\Gamma(\alpha+\beta+1)}$$
ガンマ関数の性質より $\Gamma(\alpha+1) = \alpha\Gamma(\alpha)$、$\Gamma(\alpha+\beta+1) = (\alpha+\beta)\Gamma(\alpha+\beta)$ を代入して約分すると、
$$E[p] = \frac{\alpha}{\alpha+\beta}$$

### 💡 準1級合格のためのワンポイント
* **積率母関数（MGF）のマスター必須**: 
    第6章の範囲では「和の分布」「変数変換」の証明においてMGFが最強のツールとなります。正規分布、指数（ガンマ）分布、カイ二乗分布のMGFの形と、それを微分して期待値・分散を出す操作は白紙から実行できるようにしておきましょう。
* **条件付き分布と混合分布の混同に注意**: 
    「条件付き期待値」は回帰分析などの予測モデルの基礎となる一方、「混合分布の期待値・分散」は集団が混ざった時の全体のばらつき（群内分散＋群間分散）を表します。2変量正規分布の公式（回帰直線）と混合分布の分散の公式は、記述試験で確実に点を取れるよう暗記推奨です。
