# 統計検定準１級 概要

## 出題全範囲
統計検定準1級の出題範囲は、公式テキスト「統計学実践ワークブック（以下、WB）」の全32章分に相当し、内容の関連性から大きく以下の4つの構造に分類されます。

1. **確率と確率分布**
   - 事象と確率
   - 確率分布と母関数 / 分布の特性値
   - 変数変換
   - 離散型分布 / 連続型分布と標本分布
   - 極限定理、漸近理論
2. **統計的推測**
   - 統計的推定の基礎 / 区間推定
   - 検定の基礎と検定法の導出
   - 正規分布に関する検定 / 一般の分布に関する検定法
   - ノンパラメトリック法
3. **多変量解析法**
   - 重回帰分析 / 回帰診断法 / 質的回帰 / 回帰分析その他
   - 主成分分析 / 判別分析 / クラスター分析
   - 因子分析・グラフィカルモデル / その他の多変量解析手法
4. **種々の応用**
   - マルコフ連鎖 / 確率過程の基礎
   - 分散分析と実験計画法
   - 標本調査法
   - 時系列解析
   - 分割表 / 不完全データの統計処理
   - モデル選択 / ベイズ法 / シミュレーション

## 各単元の概要
上記4つの大項目について、キーワードと概要をまとめました。

### 1. 確率と確率分布
**キーワード:**
期待値・分散・歪度・尖度、モーメント母関数、ヤコビアン、各種確率分布（二項、ポアソン、幾何、ガンマ、ベータ分布など）、中心極限定理

**概要:**
データ解析の土台となる確率論の基本から、各種確率分布の性質までを深く問われます。離散型・連続型を問わず、確率（密度）関数から期待値、分散、母関数を導出する能力が求められます。変数変換（重積分とヤコビアン）や極限定理を利用した近似計算は頻出であり、各分布の特性（再生性や無記憶性など）の違いを体系的に暗記・理解しておく必要があります。

### 2. 統計的推測
**キーワード:** 不偏推定量、一致推定量、最尤法、信頼区間、ネイマン・ピアソンの補題、尤度比検定、ウィルコクソンの順位和検定

**概要:**
母集団のパラメータを標本データから推定・検定する数理的な背景を学びます。単なる公式の適用だけでなく、不偏推定量と一致推定量の定義の違いや、検定の「最強力性」といった理論的根拠の理解が必須です。また、正規分布を前提としないノンパラメトリック手法の特徴についても問われ、データに応じた適切な推測手法を選択する力が評価されます。

### 3. 多変量解析法
**キーワード:** 最小二乗法、マルチコ、ロジスティック回帰、固有値・固有ベクトル、主成分分析（寄与率）、判別分析、マハラノビス汎距離

**概要:**
複数の変数間の関係性を数理モデル化し、次元削減や分類を行う手法群です。行列計算などの線形代数の知識が不可欠となります。重回帰分析における回帰係数や分散分析表の解釈、主成分分析における固有値計算などは実務とも直結しており、CBT形式においても数値計算問題として頻出する重要な単元です。

### 4. 種々の応用
**キーワード:** 推移確率行列、定常分布、分散分析（一元・二元配置）、AR/MAモデル、自己相関、MCMC法、事後分布

**概要:**
時間依存データや不完全データなど、より現実に近い複雑なデータを扱うための応用手法です。マルコフ連鎖における定常分布の計算や、時系列解析におけるモデルの性質は特有のアプローチを要します。また、現代データサイエンスで多用されるベイズ推定法やシミュレーション手法についても基礎的なアルゴリズムの理解が問われます。


## 単元別の優先順位と出題傾向
統計学実践ワークブック（WB）の全32章について、CBT形式における「出題頻度」と、Web Appでの問題自動生成やUI実装のしやすさを示す「アプリ相性」を3段階（〇: 3点, △: 2点, ✕: 1点）で評価し、その合計点から「総合評価」を算出しました。

| 章のタイトル | 出題頻度 | アプリ相性 | 総合評価 | 出題内容（単元内容 × CBT傾向） |

| 1 事象と確率 | 〇 | 〇 | 高 (6点) | 条件付き確率やベイズの定理など基礎的な確率計算が、ベン図や表を用いた問題として頻出します。 |
| 2 確率分布と母関数 | △ | △ | 中 (4点) | 確率母関数やモーメント母関数の定義式を用いた、期待値や分散の数理的な導出過程が問われます。 |
| 3 分布の特性値 | △ | 〇 | 高 (5点) | 共分散や相関係数の計算、および歪度・尖度の定義式に関する理解が計算・選択問題で出題されます。 |
| 4 変数変換 | ✕ | △ | 低 (3点) | ヤコビアンを用いた確率密度関数の変換式を選ぶなど、複雑な計算を避けた概念選択問題が出ます。 |

## AIが提案する学習計画（Web App開発向け）

上記の総合評価（高・中・低）の構造を活かし、スマホアプリを通じた最も効率的な学習・実装計画を提案します。

### ① アプリ実装・学習のロードマップ
ユーザーが挫折せず、かつスコアを最速で伸ばせるよう、実装順序と学習パスを一致させます。
1. **フェーズ1: コア計算の反復（総合評価 6点の章）**
   - 対象: 第5, 6, 16, 20章
   - 内容: 合否を分ける超頻出の計算問題。パラメータをランダム生成するアルゴリズムを組み、「分布の期待値・分散計算」「分散分析表（ANOVA）の穴埋め」を無限に解けるようにします。
2. **フェーズ2: 定型アルゴリズムと暗記（総合評価 5点の章）**
   - 対象: 第1, 3, 9, 10, 11, 12, 14, 22, 27, 28章
   - 内容: パターン化された検定計算やマルコフ連鎖、時系列モデルの自己相関などを実装。ここを極めればCBT形式の合格ライン（60点）を越えられます。
3. **フェーズ3: 概念理解と周辺知識（総合評価 4点以下の章）**
   - 対象:
     - 総合評価 4点の章
       - 第2, 8, 21, 23, 25, 26, 31章
     - 総合評価 3点の章
       - 第4, 7, 13, 17, 18, 19, 24, 30, 32章
     - 総合評価 2点の章
       - 第15, 29章
   - 内容: アプリのUXを損なう複雑な数式計算は避け、用語の定義、公式の形、各手法の適用条件を問う「一問一答（○×・4択）」としてサクッと学習できるようにします。

### ② 出題形式とUI/UXの設定
CBT形式に最適化した回答インターフェースを設定します。
* **数値入力モード:** テンキー型のソフトキーボードを用意。電卓を用いた計算結果を直接入力させます（第9章 区間推定、第28章 分割表のオッズ比など）。
* **5肢選択モード:** 長い数式や文章は避け、選択肢を大きめのボタンとして配置。スマホのタップで直感的に解答できるようにします。
* **統計数値表のポップアップ:** 第6章や第11章の検定問題では、「正規分布表」や「t分布表」を画面端のアイコンからいつでもオーバーレイ表示できる機能を実装します。

### ③ 類題生成エンジン（動的パラメーター機能）
「アプリ相性」が高い章を中心に、以下の変数を乱数化するだけで無数の類題を自動生成します。
* **確率分布（第5, 6章）:** サンプルサイズや成功確率をランダム化。
* **クロス集計表（第12, 28章）:** セル内の度数をランダム生成し、カイ二乗値やオッズ比を逆算。
* **分散分析表（第16, 20章）:** 平方和や自由度の数値をランダム化し、F値を算出。

### ④ 学習継続（リテンション）の仕組み
* **忘却曲線ベースの「今日の復習」:** 過去の正答率と解答時間から、定着していない問題を自動抽出し、毎日10問のデイリーミッションとして提示します。
* **単元別レーダーチャート:** 32章を「確率分布」「推定・検定」「多変量解析」「応用手法」の4象限にまとめ、自身の総合評価（習熟度）を視覚化します。
* **アンロック要素:** 総合評価【高】の単元で一定スコアを獲得すると、総合評価【低】の難解な章の「チートシート画像（要点まとめ）」がダウンロード可能になるゲーミフィケーションを導入します。
